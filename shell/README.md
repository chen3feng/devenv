# Shell Functions

See comments in each file for details.

| File | Description |
| --- | --- |
| [alt_screen](alt_screen) | Switch to the terminal alternative screen |
| [compare_version](compare_version) | Compare two dot-separated version strings |
| [find_sources](find_sources) | Find all files with specified file extensions |
| [macos](macos) | macOS helpers: `unquarantine`, `allowapp` (codesign + unquarantine), `disablesleep`/`restoresleep` |
| [path_manager](path_manager) | Manage colon-separated variables (PATH, LD_LIBRARY_PATH, etc.) |
| [proxy](proxy) | Toggle `http_proxy`/`https_proxy`/`all_proxy` on or off |
| [trash_rm](trash_rm) | Replace `rm` with a trash-aware version (macOS Finder trash on Darwin, `~/.local/trash` on Linux) |
